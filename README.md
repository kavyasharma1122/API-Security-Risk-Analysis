# API Security Risk Analysis

This project contains an API security risk analysis performed on a demo API.

## API Tested
JSONPlaceholder Test API

## Tools Used
- Postman
- Browser DevTools

## Scope
Read-only API security inspection.

## Methodology
1. Reviewed API documentation
2. Tested endpoints using Postman
3. Inspected request headers and responses
4. Identified possible security risks

## Risks Identified
- Open API endpoints
- Excessive data exposure
- Missing rate limiting
- Weak input validation
